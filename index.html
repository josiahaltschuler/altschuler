BlueAlly | Web application to manage Verizon's VoIP networks | Angular, PHP (Slim) API, Sass

A large-scale web application to manage all of Verizon's VoIP networks. The web application includes business rules for automatically generating Method of Procedures (MOPs) and configuration files, which can then be applied to infrastructure devices such as routers, firewalls and switches. It also validates data across multiple sites to eliminate data errors, such as conflicting information, invalid data and duplicate settings. The Verizon Portal provides additional benefits in that users can provision downstream systems, as well as audit production sites for differences between production and the Verizon Portal “database of record.” Customizable reports are available to allow users to query information from multiple sites for real-time analysis.
